### git 规范

**git 分支规范**：

**git commit message规范**：

- feat(feature): 添加了新特性。
- fix(bug): 修复了 bug。
- hotfix: 补丁包。
- chore: 修改了些单词，错别字。
- test: 添加/修改了测试
- docs: 添加/修改了文档。
- merge: 从其它分支合并过来。
- revert: 回退。

[Git Commit 完整规范]: https://www.ruanyifeng.com/blog/2016/01/commit_message_change_log.html
